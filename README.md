## FC
一款简单的 typecho 主题

预览：[ Jstar ](https://blog.joessem.com/)

-------

## 使用

1. 下载此项目源码
2. 解压，上传 ` FC` 文件夹至网站 Typecho 根目录下的 `user/themes/` 路径
3. 到网站后台 - 外观 - 主题 中启用此主题
4. 设置外观



## 功能

请查看主题文章： [FC](https://blog.joessem.com/index.php/archives/FC.html)
